Açık Kuran API

Open-source, ad-free Qur'an API providing verses, translations, word-by-word analysis, and more.

GitHub acikkuran.com CC BY-NC-SA 4.0
Base URL https://api.acikkuran.com

Authors

GET
/authors
Lists all translation authors.

Example: https://api.acikkuran.com/authors 

{
  "data": [
    { "id": 50, "name": "Erhan Aktaş", "description": "Kerim Kur'an", "language": "tr", "url": null },
    { "id": 102, "name": "The Monotheist Group", "description": "The Quran: A Monotheist Translation", "language": "en", "url": null }
  ]
}

Surahs

GET
/surahs
Lists all 114 surahs (chapters).

Example: https://api.acikkuran.com/surahs 

{
  "data": [
    {
      "id": 1, "name": "Fatiha", "name_en": "Al-Fatihah", "slug": "fatiha",
      "verse_count": 7, "page_number": 0, "name_original": "سُورَةُ ٱلْفَاتِحَةِ",
      "audio": { "mp3": "https://audio.acikkuran.com/tr/1.mp3", "duration": 38 }
    }
  ]
}
GET
/surah/[surah_id]?author=[author_id]
Gets surah details and lists all its verses with translations.

Example 1 (default author): https://api.acikkuran.com/surah/1

Example 2 (specific author): https://api.acikkuran.com/surah/6?author=8

Verses

GET
/surah/[surah_id]/verse/[verse_number]?author=[author_id]
Gets verse detail with translation.

Example: https://api.acikkuran.com/surah/6/verse/1

GET
/surah/[surah_id]/verse/[verse_number]/translations
Lists all available translations for a verse.

Example: https://api.acikkuran.com/surah/6/verse/1/translations

GET
/surah/[surah_id]/verse/[verse_number]/verseparts
Lists word-by-word breakdown (verseparts) for a verse.

Example: https://api.acikkuran.com/surah/6/verse/1/verseparts

Pages

GET
/page/[page_number]?author_id=[author_id]
Gets all verses on a given Qur'an page with translations.

Example: https://api.acikkuran.com/page/1

Roots

GET
/root/latin/[latin_chars]
Gets root word information by its Latin transliteration.

Example: https://api.acikkuran.com/root/latin/ktb

GET
/root/latin/[latin_chars]/verses?page=[page]&author=[author_id]
Lists verses containing words from a given root.

Example: https://api.acikkuran.com/root/latin/ktb/verses

GET
/root/latin/[latin_chars]/verseparts?page=[page]&author=[author_id]
Lists verseparts (words) derived from a given root.

Example: https://api.acikkuran.com/root/latin/ktb/verseparts

GET
/root/[id]
Gets a root word by its numeric ID.

Example: https://api.acikkuran.com/root/1

GET
/rootchars
Lists all root characters (first letters) available.

Example: https://api.acikkuran.com/rootchars

GET
/rootchar/[id]
Lists all roots starting with a given root character.

Example: https://api.acikkuran.com/rootchar/1

Search

GET
/search?q=[query]&author=[author_id]
Full-text search across translations (requires MeiliSearch).

Example: https://api.acikkuran.com/search?q=rahman
